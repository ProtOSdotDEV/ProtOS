Quick Start

# Set up the repository
protos:/# protpkg repo https://YOUR_USERNAME.github.io/protos-repo

# Sync the package index
protos:/# protpkg sync
:: Synchronizing package index...
>>> Package index synced

# Install a package
protos:/# protpkg install nano

Commands

protpkg sync # Update package index
protpkg install <pkg> # Install a package
protpkg remove <pkg> # Remove a package
protpkg search <term> # Search packages
protpkg list # List installed packages
protpkg upgrade # Upgrade all packages
protpkg info <pkg> # Show package details

About

All packages in this repository are compiled from source and statically linked for ProtOS. No dynamic libraries required — every binary is self-contained. Packages are built for aarch64 and verified with SHA-256 checksums.
